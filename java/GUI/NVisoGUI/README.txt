This application provides a GUI interface to the nViso 3D Facial Imaging API that allows a folder of images to be processed and a CSV file to be generated.

To run the application you will the Java JRE 1.7 or higher installed. 
http://www.oracle.com/technetwork/java/javase/downloads/jre7-downloads-1880261.html

To launch the application please follow the following steps :

a. Open a command window.
b. Type> java.exe -jar "NVisoAPIGui.jar" __INSERT_YOUR_APP_ID__  __INSERT_YOUR_APP_KEY__

To use the application please select a folder to process. The application will search all image files in that folder with extension .jpg, .jpeg, or .png. It will then process the images and you can export the results as a CSV file.
